# Unity Essentials Study

Unity Learn의 **Unity Essentials Pathway** 실습 프로젝트입니다.  
Unity 엔진의 기본 사용법과 게임 제작의 핵심 개념(씬 구성, 오브젝트, 머티리얼, 조명, 스크립트 등)을 학습했습니다.

## 학습 내용
- Unity 에디터 기본 조작 (Hierarchy, Inspector, Project, Scene View)
- 3D 오브젝트 생성 및 머티리얼 적용
- 기본 조명(Light)과 카메라 설정
- Transform과 컴포넌트 개념 이해
- 간단한 C# 스크립트 작성 및 연결
- 빌드 및 실행 테스트

## 프로젝트 정보
- Unity 버전: 6.2 (또는 6.2 등 실제 사용 버전)
- Pathway: [Unity Essentials](https://learn.unity.com/pathway/unity-essentials)
- 목적: Unity 기초 학습 및 실습 기록용

## 참고
이 프로젝트는 Unity Learn 공식 튜토리얼을 기반으로 한 학습용 샘플입니다.
상업적 목적이 없으며, 코드 및 에셋은 학습을 위해 사용되었습니다.
